Add numbers

Please attach a CSV file (you can save an excel spreadsheet as a CSV file).

If you've already uploaded a list of contacts, new contacts will be appended to the end of the existing list (a new CSV upload does NOT replace your existing uploaded contacts).

Allowed fields (case-insensitive) in your spreadsheet are:

'Number' - The phone number.

'Company' - Name of company to contact.

'Address' - Company address.

'Name' - Name of person the agent will be calling if available.

'Role' - Job title of person the agent will be calling.

'Email' - Email address of person the agent will be calling.

'Notes' - For anything about this contact that the agent should be aware of.

'Outcome' - Yak Hub will update this field once the campaign is live.

'Rating' - Yak Hub will update this field once the campaign is live.

'Called' - This has a value of 'Y' or 'N' to denote if the number's been dialled already.
